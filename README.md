## 프로젝트 소개
- 이름 : MITI
- 소개 : 사용자에게 개인 맞춤형 음악을 추천 기능을 제공하는 음악 스트리밍 서비스
- 기간: 2024.02-2024-11
- 환경 : SpringBoot, Java, Python, JavaScript, MySql, Spotify Api
- <a href="https://miti.p-e.kr"/>https://miti.p-e.kr</a>
<br>

## 주요 기능
- Spotify 소셜 로그인
- 음악 스트리밍
- 유사한 음악 추천
- 맞춤형 음악 추천
- 장르별 음악 추천
- 곡 평가
